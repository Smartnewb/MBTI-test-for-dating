# 달빛 연애 연구소 - MBTI 연애 테스트 기획서

## 📌 서비스 개요

- *「달빛 연애 연구소」**는 Z세대를 위한 MBTI 기반 연애 심리 테스트로, 사용자의 연애 스타일을 분석하고 가장 잘 맞는 이상형의 MBTI를 제안합니다. 테스트 결과는 공유하기 쉬운 형태로 제공되어 자연스러운 바이럴을 유도하고, 최종적으로 썸타임 대학생 소개팅 서비스로의 가입을 유도합니다.

### 🎯 핵심 목표

- Z세대의 연애 심리 및 성향을 바탕으로 바이럴 요소가 강한 MBTI 테스트 제공
- "공유하고 싶은 결과"를 설계하여 **SNS 자발적 확산 유도**
- 썸타임 브랜드 인지도 향상 및 서비스 사전신청 유도 (CTA 포함)

## 💫 서비스 차별화 포인트

1. **깊이 있는 연애 심리 분석**: 일반 MBTI가 아닌 연애 맥락에 특화된 질문으로 정확한 성향 파악
2. **이상형 매칭 시스템**: 단순 성향 분석을 넘어 궁합이 좋은 MBTI 유형 추천
3. **시각적 공유 요소**: 인스타그램, 카카오톡에서 공유하기 좋은 결과 카드 디자인
4. **보라색 타로 컨셉**: 신비로운 분위기로 호기심과 몰입감 유도

## 🛠 서비스 구조 및 기능

### 1. 인트로 페이지

**디자인 & UX 요소:**

- 썸타임 로고 & 별 일러스트 배경 (상단)
- "지금 너의 연애 스타일은 어떤 모습일까?" 질문 텍스트 (중앙)
- 보라색 타로카드 일러스트 (중앙)
- "테스트 시작하기" CTA 버튼 (하단)

### 2. 테스트 문항 페이지

**구조:**

- MBTI 4가지 축: E/I, S/N, F/T, J/P
- 축당 6문항 (기본 성향 2 + 연애 성향 4)
- 총 24문항, 2지선다형 선택

**디자인 & UX 요소:**

- 진행률 표시 (상단): "5/24"
- 질문 텍스트 + 2개 선택지 카드 형태 (중앙)
- 선택 시 자동 다음 질문으로 슬라이드 전환 (애니메이션)

### 3. 결과 페이지

**구조:**

1. **나의 MBTI 연애 스타일 요약**
    - "[MBTI 유형] 연애 스타일입니다" 헤드라인
    - 해당 유형의 연애 성향 요약 설명
2. **이상형 MBTI 추천 및 궁합 해설**
    - "당신에게 가장 잘 맞는 유형은 [이상형 MBTI]입니다"
    - 이상형 유형의 연애 성향 요약
    - 궁합이 잘 맞는 이유에 대한 심리적 해설
3. **연애 키워드 요약**
    - 연애 장점 | 연애 단점 | 갈등 대응 | 잘 맞는 유형 특징
4. **CTA 버튼**
    - "내 연애 이상형 결과 공유하기" (SNS 공유)
    - "나랑 어울리는 사람, 썸타임에서 만나볼래요?" (썸타임 연결)

## 📋 테스트 문항 구성 (MBTI 4축 x 6문항)

- 총 24개 문항으로 구성
- 각 MBTI 축(E/I, S/N, F/T, J/P)별로 6개 문항
- 각 문항은 2개의 선택지 제공
- 연애 상황에 특화된 질문으로 구성

## 🎭 결과 페이지 컨텐츠

### 1. MBTI 유형별 연애 성향 요약
- 16가지 MBTI 유형별 연애 스타일 설명
- 각 유형의 특징적인 연애 행동 패턴 설명

### 2. 이상형 MBTI 궁합 해설
- 사용자의 MBTI 유형과 가장 잘 맞는 이상형 MBTI 제안
- 궁합이 좋은 이유에 대한 심리적 설명

### 3. 궁합 주의보
- 사용자의 MBTI와 궁합이 좋지 않은 유형 경고
- 궁합이 좋지 않은 이유 설명

### 4. 연애 키워드 요약
- 연애 장점
- 연애 단점
- 갈등 대응 방식
- 잘 맞는 유형의 특징
